# The future of ferushtml
ferushtml will most likely get phased out now, or atleast until ferus ceases to be a one-man project.

# Why?
I've been very busy recently and I hardly have any time to allocate for Ferus. I've decided that it's better to make one of the harder parts of writing a browser -- the HTML parser alone. `feruscss` will most likely come into existence soon though as it is much less of a pain to parse. I mostly work in writing renderers due to my game-dev experience hence I don't write good parsers for really complex stuff. ferushtml can parse simple HTML, but thanks to Microsoft Internet Explorer we have the mess that we today call HTML. I might start contributing to `chame` if I get time in order to help them out.
